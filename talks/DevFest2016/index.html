AHEAD-OF-TIME COMPILATION (AoT)

  • Compile templates at build before serving to client
  • Fast rendering
  • Less requests
  • Smaller framework size
  • Detect errors at build
  • Better security

Large applications and TypeScript : superset of javascript (es6/7 + Types)

  • Created by Anders Hejlsberg (c#, Delphi, Turbo Pascal) .
  • Compile from TS => Javascript : ES5, ES6.
  • Plain javascript in runtime.
  • Predictable data flow.
  • Features from the future, es6, es7, and even es8 !
  • For IDEs : Enable the power for code completion, instant errors, code refactoring, go to definition.
